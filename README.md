## Overview

`project2026` is an R package that answers three research questions about a
dataset of 1,000 adults aged 26–45:

| # | Research Question | Test Used |
|---|---|---|
| 1 | Is there a linear relationship between height and weight? | Simple linear regression (`lm()`) |
| 2 | Is the average height the same for males and females? | Two-sample *t*-test, equal variances (`t.test()`) |
| 3 | Is there any association between gender and physical activity? | Chi-square test of independence (`chisq.test()`) |

Each wrapper function:
- states the null and alternative hypotheses,
- produces numerical and graphical assumption checks,
- fits the appropriate model and calculates the test statistic,
- outputs the decision at a chosen significance level (default α = 0.05), and
- provides a plain-English conclusion.

All three functions return a structured named list with a custom S3 `print()`
method.

## Quick Start

```r
library(project2026)

# Load the bundled dataset
data(project2026_data)
head(project2026_data)

# ── Research Question 1: Linear Regression ───────────────────────────────────
rq1 <- linear_reg(project2026_data)

# Access individual components
rq1$decision        # "Reject H0" or "Fail to reject H0"
rq1$conclusion      # plain-English sentence
rq1$r_squared       # R-squared value
rq1$plots$scatter   # ggplot scatter with regression line
rq1$plots$residuals # ggplot residuals vs fitted
rq1$plots$qq        # ggplot normal Q-Q plot

# ── Research Question 2: Two-Sample t-test ───────────────────────────────────
rq2 <- my_ttest(project2026_data)

rq2$group_summary   # per-gender means and SDs
rq2$test_statistic  # t-statistic
rq2$conf_int        # 95% CI for mean difference
rq2$decision
rq2$conclusion

# ── Research Question 3: Chi-Square Test ─────────────────────────────────────
rq3 <- my_chi_square(project2026_data)

rq3$observed        # contingency table of observed counts
rq3$expected        # expected counts under H0
rq3$min_expected    # assumption check: should be >= 5
rq3$decision
rq3$conclusion

# ── Suppress plots (useful inside Quarto reports) ────────────────────────────
rq1 <- linear_reg(project2026_data,    alpha = 0.05, plot = FALSE)
rq2 <- my_ttest(project2026_data,      alpha = 0.05, plot = FALSE)
rq3 <- my_chi_square(project2026_data, alpha = 0.05, plot = FALSE)
```

## Dataset

`project2026_data` — 1,000 adults aged 26–45.

| Column | Type | Description |
|---|---|---|
| `ID` | character | Participant ID (e.g. `"ID1"`) |
| `gender` | character | `"Male"` or `"Female"` |
| `height` | numeric | Height in centimetres (cm) |
| `weight` | numeric | Weight in kilograms (kg) |
| `phys` | character | Physical activity: `"None"`, `"Moderate"`, `"Intense"` |

> **Note:** In the raw CSV, the `"None"` physical activity category is stored
> as `NA`. The `data-raw/project2026_data.R` script and all wrapper functions
> recode `NA` to `"None"` automatically.
